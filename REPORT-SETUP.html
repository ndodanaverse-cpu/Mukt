📋 Report Modal Setup Guide

Step 1: Add Report Modal HTML

Add this HTML code to your pages (e.g., customer-home.html, customer-vendor-profile.html, etc):

<!-- Report Modal --> <div id="reportModal" class="modal" style="display:none;"> <div class="modal-content"> <div class="modal-header"> <h2>📋 Report <span id="reportTarget"></span></h2> <button class="modal-close" onclick="closeReportModal()">×</button> </div> <div class="modal-body"> <div class="form-group"> <label for="reportReason">Reason for Report</label> <select id="reportReason"> <option value="">-- Select Reason --</option> <option value="Inappropriate Content">Inappropriate Content</option> <option value="Scam/Fraud">Scam/Fraud</option> <option value="Poor Quality">Poor Quality</option> <option value="False Information">False Information</option> <option value="Offensive Language">Offensive Language</option> <option value="Other">Other</option> </select> </div> <div class="form-group"> <label for="reportDetails">Additional Details</label> <textarea id="reportDetails" placeholder="Please provide more information..." rows="4"></textarea> </div> </div> <div class="modal-footer"> <button class="btn btn-secondary" onclick="closeReportModal()">Cancel</button> <button class="btn btn-danger" onclick="submitReport()">Submit Report</button> </div> </div> </div>

Step 2: Add Modal CSS

Add this CSS code to your <style> section:

.modal { position: fixed; top: 0; left: 0; width: 100%; height: 100%; background: rgba(0,0,0,0.7); display: flex; align-items: center; justify-content: center; z-index: 9999; } .modal-content { background: white; border-radius: 12px; max-width: 500px; width: 90%; box-shadow: 0 20px 60px rgba(0,0,0,0.3); } .modal-header { display: flex; justify-content: space-between; align-items: center; padding: 20px; border-bottom: 1px solid #e5e7eb; } .modal-header h2 { font-size: 18px; margin: 0; } .modal-close { background: none; border: none; font-size: 28px; cursor: pointer; color: #9ca3af; } .modal-body { padding: 20px; } .modal-footer { display: flex; gap: 10px; padding: 20px; border-top: 1px solid #e5e7eb; justify-content: flex-end; } .form-group { margin-bottom: 15px; } .form-group label { display: block; font-weight: 600; margin-bottom: 6px; color: #374151; font-size: 14px; } .form-group select, .form-group textarea { width: 100%; padding: 10px; border: 1px solid #e5e7eb; border-radius: 6px; font-family: 'Inter', sans-serif; font-size: 14px; } .form-group select:focus, .form-group textarea:focus { outline: none; border-color: #1a1a2e; box-shadow: 0 0 0 3px rgba(26,26,46,0.1); }

Step 3: Add Report Buttons

Add report buttons wherever needed. Examples:

For Vendors (on customer-vendor-profile.html):

<button onclick="openReportModal('vendor', vendorId, vendorName)">🚩 Report Vendor</button>

For Vendors (on vendor cards):

<button onclick="openReportModal('vendor', vendorId, vendorName)" style="margin-top: 10px;">Report</button>

For Customers (user profile or chat):

<button onclick="openReportModal('customer', customerId, customerEmail)">🚩 Report User</button>

Step 4: Add JavaScript Functions

Add these functions to your page's <script> section:

let currentReport = { type: null, id: null, name: null }; function openReportModal(type, id, name) { currentReport = { type, id, name }; document.getElementById('reportTarget').textContent = type === 'vendor' ? `Vendor: ${name}` : `User: ${name}`; document.getElementById('reportModal').style.display = 'flex'; document.getElementById('reportReason').value = ''; document.getElementById('reportDetails').value = ''; } function closeReportModal() { document.getElementById('reportModal').style.display = 'none'; } async function submitReport() { const reason = document.getElementById('reportReason').value; const details = document.getElementById('reportDetails').value; if (!reason) { alert('Please select a reason'); return; } try { const user = firebase.auth().currentUser; if (!user) { alert('Please login to submit a report'); return; } await firebase.firestore().collection('reports').add({ reportType: currentReport.type, vendorId: currentReport.type === 'vendor' ? currentReport.id : null, vendorName: currentReport.type === 'vendor' ? currentReport.name : null, customerId: currentReport.type === 'customer' ? currentReport.id : null, customerEmail: currentReport.type === 'customer' ? currentReport.name : null, reason: reason, details: details, reporterEmail: user.email, reporterUid: user.uid, createdAt: new Date(), status: 'pending' }); alert('✅ Report submitted successfully. Admin will review soon.'); closeReportModal(); } catch (err) { console.error('Report error:', err); alert('❌ Error submitting report: ' + err.message); } } // Close modal when clicking outside document.addEventListener('click', function(e) { const modal = document.getElementById('reportModal'); if (e.target === modal) { closeReportModal(); } });
💡 Quick Insert Guide:

To quickly add report functionality to a page:

  1. Scroll to the bottom of your HTML file's <body>
  2. Paste the HTML from Step 1 before </body>
  3. Add the CSS from Step 2 to your <style> section
  4. Add the JS from Step 4 to your <script> section
  5. Add report buttons using examples from Step 3
🔗 Important:

Make sure your page has the Firebase SDK loaded (already has it in your existing pages)
